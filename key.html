How to Obtain An API Key

1. Create An Account

If you do not already have an account, you can create one here.

2. Purchase Keys

An API key is good for 10,000 requests, but an unlimited number of API keys can be purchased for any account. TODO: actual purchasing

You will eventually recieve your key via email. Note, use the same email for the user purchase as the paypal account.

3. Use A Key

API keys can be used in place of session cookies for the REST functions. Simply send a cookie of ID=YOUR_API_KEY to verify you requests.
